🧭 How to Install

  1. Click the Download button above to get the ZIP file from Google Drive.
  2. Extract the extension.zip file to a folder.
  3. Open your browser and go to chrome://extensions/.
  4. Enable Developer mode (top right corner).
  5. Click “Load unpacked” and select the extracted folder.
  6. The News Fact Checker icon will appear in your toolbar!

⚙️ How It Works

This extension helps users verify the authenticity of online news using a combination of APIs:

  • ✅ Google Fact Check API — checks claims and fact-check sources.
  • 🎥 YouTube Data Captions API — analyzes captions of videos for misinformation.
  • 🧠 AI Image Analyzer — detects if an image is AI-generated.
  • 🌐 Google Custom Search JSON API — finds credible articles for comparison.

The extension highlights suspicious news articles and provides verified sources and context — helping you quickly separate fact from fiction.
